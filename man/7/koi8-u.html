NAME

koi8-u - Ukrainian character set encoded in octal, decimal, and hexadecimal

DESCRIPTION

RFC 2310 defines an 8-bit character set, KOI8-U. KOI8-U encodes the characters used in Ukrainian and Byelorussian.
