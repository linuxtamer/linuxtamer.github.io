NAME

cp1252 - CP 1252 character set encoded in octal, decimal, and hexadecimal

DESCRIPTION

The Windows Code Pages include several 8-bit extensions to the ASCII character set (also known as ISO 646-IRV). CP 1252 encodes the characters used in many West European languages.
