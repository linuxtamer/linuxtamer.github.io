NAME

qt5ct - Qt5 Configuration Utility

SYNOPSIS

qt5ct

DESCRIPTION

qt5ct is utility allows users to configure Qt5 settings (theme, font, icons, etc.) under DE/WM without Qt integration.

Qt5ct could use qt5 gtk theme, KDE breeze theme, the fusion theme, QtCurve theme and many more. Each style could be customized further by changing the color schema.

OPTIONS

qt5ct hasn't options. It just works.
